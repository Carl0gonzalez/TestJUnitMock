# ✅ TestJUnitMock

Proyecto de pruebas unitarias con **JUnit y mocking** para validar lógica de negocio en Java.

![Java](https://img.shields.io/badge/Java-blue?logo=openjdk) ![JUnit](https://img.shields.io/badge/JUnit-testing-25A162?logo=junit5)

## Descripción

Está orientado a practicar aislamiento de dependencias y escritura de pruebas más controladas mediante dobles o mocks.

## Objetivo

Reforzar testing unitario en Java aplicando técnicas de simulación de dependencias.

## Uso

```bash
mvn test
```
