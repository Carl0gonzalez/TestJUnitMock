# ✅ TestJUnitMock

Proyecto de pruebas unitarias con **JUnit y mocking**, orientado a validar lógica de negocio mediante tests aislados.

![Java](https://img.shields.io/badge/Java-blue?logo=openjdk) ![JUnit](https://img.shields.io/badge/JUnit-testing-25A162?logo=junit5)

---

## 📌 Objetivo

Practicar pruebas unitarias en Java y uso de objetos simulados o dobles de prueba para aislar dependencias.

## 🧩 Qué aporta

- Testing sobre lógica de negocio.
- Separación entre comportamiento esperado y dependencias externas.
- Base para reforzar calidad de software en Java.

## ▶️ Ejecutar

```bash
git clone git@github.com:Carl0gonzalez/TestJUnitMock.git
cd TestJUnitMock
mvn test
```
